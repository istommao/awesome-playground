## Data Type

```lua
-- string
print(type("string"))
print(type(type(1)))

-- number
print(type(1))

-- table
print(type({}))

-- boolean
print(type(true))

-- function
print(type(print))
print(type(type))

-- nil
print(type(nil))
```
